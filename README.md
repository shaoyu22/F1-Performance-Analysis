# 🏎️ F1 Technical Performance Analysis: Suzuka 2024

This project analyzes Formula 1 telemetry data using Python and FastF1 API to evaluate driver performance and technical mastery.

## 📂 Project Structure
- **/notebooks**: Phase A (Sector 1 Analysis), Phase B (Weather API), Phase C (Final Insights).
- **/data**: Processed race telemetry and lap data.
- **/scripts**: Python scripts for automated data fetching.
- **/visualizations**: Performance heatmaps and tiering charts.
